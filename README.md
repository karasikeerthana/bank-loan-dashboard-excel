# Bank Loan Dashboard (Excel Data Analytics Project)

## Project Overview
This project analyzes bank loan data and presents insights using an interactive Excel dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Data Cleaning

## Key KPIs
- Total Loan Applications
- Total Funded Amount
- Total Received Amount
- Average Interest Rate
- Good Loan vs Bad Loan Percentage

## Dashboard Features
- Interactive Filters
- Dynamic KPIs
- Loan Status Analysis
- State-wise Loan Distribution
- Monthly Loan Trends

## Dataset
Financial Loan Dataset
